Student Performance Prediction system

## Description
This project analyzes student data from a CSV file to predict performance scores using a Random Forest Classifier. It includes data visualization and a machine learning pipeline.

## Requirements
You must have Python installed along with these libraries:
* pandas
* seaborn
* matplotlib
* scikit-learn

## How to Run
1. Place `ai_project.py` and `StudentPerformance.csv` in the same folder.
2. Open your terminal and run:
   ```bash
   python ai_project.py
